Nineteen Eighty-Four

Object

var nineteenEightyFour = { author: 'George Orwell', title: 'Nineteen Eighty-Four', year: 1949 }

getElementById()

getElementById() function locates DOM element by ID. In this case, a p element object is returned since the function's demo property matches the id parameter in the p element. innerHTML sets the HTML markup within the element.

<p id="demo"></p>
document.getElementById("demo").innerHTML

Type Coercion

The data type of the nineteenEightyFour.year property is 'number' which is implicitly coerced to a 'string' type.

> typeof(nineteenEightyFour) 'object' > typeof(nineteenEightyFour.year) 'number' > nineteenEightyFour.year 1949 > typeof('a string' + nineteenEightyFour.year + 'a string') 'string'

Script

var nineteenEightyFour = { author: 'George Orwell', title: 'Nineteen Eighty-Four', year: 1949 } document.getElementById("demo").innerHTML = nineteenEightyFour.author + ' published <em>' + nineteenEightyFour.title + '</em> in ' + String(nineteenEightyFour.year) + '.'
